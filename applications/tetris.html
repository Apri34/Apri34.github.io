Description

This is a tetris game developed for Android with LibGdx.

Motivation

I developed this game in order to get into the basics of game development. LibGdx was perfect for that, because it supports 2D game development.

What I learned

  • Developing simple 2D games with LibGdx
  • Using the devices accelerometer
