Unicorns



Description

This is a very simple native Android app that alows users to create and delete unicorns. Those unicorns are stored on a webserver and also locally in a database to be still able to use the app, when there is no connection.

Motivation

I developed this application due to a job interview. I was needed to learn two concepts: 'Databinding' and 'Sync Adapters'. To show the abilities I have learned during to this process, I decided to use this app to extend my portfolio.

What I learned

  • How to properly implement the Offline-First concept
  • How to implement SyncAdapters
  • The concept of MVVM and how to implement it
  • Using Googles Volley Library for HTTP requests
  • The concept of Databinding and how to implement it
  • Using GreenDao Library to create SQLite databases
