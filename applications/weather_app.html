Weather App



Description

This is a Flutter application that allows users to look up the weather everywhere in the world, including details like the weather type, temperature, wind speed and direction.

Motivation

As cross-platform frameworks continue to grow, I wanted to get a basic understanding of one of the newer frameworks. That is why I developed this application.

What I learned

  • I learned the basic concepts of Flutter
  • I learnet the basics of dart programming language
  • I learned how to perform HTTP requests in Flutter
  • I learned how to use Routing
